SETUP:
1. Download and run XAMPP
2. Enable MariaDB and Apache*
3. Put the "OMS" file into "htdocs" file in XAMPP directory
4. Open the file and edit it

* - If the Apache server does not start up, change the port number in configuration from 80 -> 82 go to XAMPP -> Config -> Apache(httpd.conf) 
	and then change all the number 80s to 82s, you can use Ctrl + F to find all port 80s faster, there are 3 to change.
